
# Codeigniter 4 API + Vue.JS




## Deployment Backend

Untuk backend install package menggunakan composer

```bash
  composer install
```

```bash
  configurasi pada .env
```
start server
```bash
  php spark serve
```

## Deployment Frontend

Untuk fronend install module terlebih dahulu

```bash
  npm install
```
start server

```bash
  npm run serve
```
